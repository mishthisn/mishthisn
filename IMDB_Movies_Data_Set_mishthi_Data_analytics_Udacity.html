Introduction¶

I have selected data set that contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue.

Here are the questions that I plan to explore

  1. Are big budget movies more popular than lower budget movies?
  2. Do big budget films generate more revenue?
  3. Are movies with longer runtime more popular?
  4. Who are the directors with most numbers of movies?
  5. What were most popular genres in each year?
  6. How many movies were made in each year?
  7. Who were most popular directors each year?
  8. Did movies in 1960s had more budgets/adjusted budgets than in 2015?
  9. Did movies in 1960s had more revenue/adjusted revenue than in 2015?

  • From above graphs, although we can see that popularity and revenue values are higher as we go to the right side of graph, it is not easy to conclude that fact.
  • It is more easier to see that co-relation when mean popularity values for each bucket are plotted.
  • Hence I wrote a generic function named plot_mean_values()
    • To take x-axis and y-axis columns as parameter
    • X Axis column name defaults to 'budget' since it is used in many graphs
    • Caller can pass its own bins and bin labels.
      • Default values for these parameters are used when budget is on X-Axis
    • X-axis column is divided into bins using cut function so that each row has a bin label.
    • Data frame is then grouped by these labels using groupby function.
    • Mean value of Y-Axis column in each bin calculated and plotted against X-Axis column

def plot_mean_values(y_axis_column, x_axis_column = 'budget', bins = None, bin_labels = None):

    # Create bins if not passed by caller
    if bins is None:  
        bins = [0, 1e6, 5e6, 10e6, 25e6, 50e6, 100e6, 150e6, 200e6]

    # Create labels for bins if not passed by caller
    if bin_labels is None:
        bin_labels = ['0-1M', '1-5M', '5-10M', '10-25M', '25-50M', '50-100M', '100-150M', '150-200M']

    # Use Pandas cut command to divide x_axis_column values in bins 
    # and create new column named x_axis_column in data frame
    # Each row in data frame will have a column x_axis_column with one of values in bin_labels
    bin_column_name = x_axis_column + '_bins'
    df[bin_column_name] = pd.cut(df[x_axis_column], bins, labels=bin_labels)

    # Define two arrays
    # bins_values will hold 8 possible values from bin_labels
    bins_values = []
    # mean_values will hold average popularity values for each bucket  
    mean_values = []

    # groupby function groups all rows together that have same values in budget_bins
    # E.g. all movies with revenue between 5-10M will be in group titled '5-10M'
    bin_groups = df.groupby(bin_column_name)

    # Iterate over groups
    # There are 8 groups here.
    # Each one of 10k+ rows are in one of 8 groups
    for group_title, group in bin_groups:
        
        # Store groups title in budget_bins_values array
        bins_values.append(group_title)
        
        # From correspoding group, extract the column, calculate mean and store in mean_values array
        mean_values.append(group[y_axis_column].mean())

    # Create Python object from two arrays  
    y_axis_column_name = "{0} mean".format(y_axis_column)
    obj_for_df = { x_axis_column : bins_values, 
                   y_axis_column_name : mean_values }

    # Create a new dataframe from python object
    popularity_df = pd.DataFrame(obj_for_df)

    # Plot graph of budget bins with
    graph_title = '{0} vs {1}'.format(y_axis_column, x_axis_column)
    popularity_df.plot(x_axis_column, y_axis_column_name, figsize=(15,4), title=graph_title)
